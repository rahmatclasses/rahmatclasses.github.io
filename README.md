# Hafiz Ji Website

This is a complete static website created for Hafiz Ji.

## Files

- `index.html` - page structure and editable text
- `styles.css` - full responsive design
- `script.js` - mobile menu and scroll reveal effects

## How to open it

1. Open `index.html` in any browser.
2. To edit text, open `index.html`.
3. To change colors and design, edit `styles.css`.
4. To update phone, WhatsApp, email, and location, edit the links in the contact section of `index.html`.
